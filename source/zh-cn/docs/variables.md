### 全局变量

| 变量    | 描述                               | 类型                        |
| ----- | -------------------------------- | ------------------------- |
| `站点`  | 全站信息                             | `object`; 见 [网站变量](#网站变量) |
| `页面`  | 针对该页面的内容以及 front-matter 中自定义的变量。 | `object`; 见 [页面变量](#页面变量) |
| `配置`  | 网站配置                             | `object` (站点的配置文件)        |
| `主题`  | 主题配置。 继承自网站配置。                   | `object` (主题配置文件)         |
| `路径`  | 当前页面的路径（不含根路径）                   | `字符串`                     |
| `网址`  | 当前页面的完整网址                        | `字符串`                     |
| `env` | 环境变量                             | ???                       |

{% note warn %}
从 Hexo 5.0.0 开始，Lodash 已从全局变量中移除。 迁移时 [You-Dont-Need-Lodash-Underscore](https://github.com/you-dont-need/You-Dont-Need-Lodash-Underscore) 或许能为你提供帮助。
{% endnote %}

### 网站变量

| 变量   | 描述   | 类型                          |
| ---- | ---- | --------------------------- |
| `帖子` | 所有文章 | `array`，包含了站点全部的文章 `object` |
| `页面` | 所有分页 | `array`，包含了站点全部的分页 `object` |
| `分类` | 所有分类 | `array`，包含了站点全部的分类 `object` |
| `标签` | 所有标签 | `object` 或 `null`           |

### 页面变量

**页面（`page`）**

| 变量           | 描述                                            | 类型               |
| ------------ | --------------------------------------------- | ---------------- |
| `page.title` | 文章标题                                          | `字符串`            |
| `日期`         | 文章创建日期                                        | [Moment.js][] 对象 |
| `更新页面`       | 页面更新日期                                        | [Moment.js][] 对象 |
| `评论`         | 评论已启用或未启用                                     | `boolean`        |
| `页面布局`       | 布局名称                                          | `字符串`            |
| `内容`         | 页面的完整内容                                       | `字符串`            |
| `摘要页面`       | 文章摘要                                          | `字符串`            |
| `更多`         | 除了页面摘要的其余内容                                   | `字符串`            |
| `源文件`        | 源文件的路径                                        | `字符串`            |
| `完整页面`       | 页面的完整原始路径                                     | `字符串`            |
| `路径`         | 页面网址（不含根路径）。 我们通常在主题中使用 `url_for(page.path)`。 | `字符串`            |
| `永久链接`       | 文章的完整(加密) URL                                 | `字符串`            |
| `上一页`        | 上一个页面。 如果此为第一个页面则为 `null`。                    | ???              |
| `下一页`        | 下一个页面。 如果此为最后一个页面则为 `null`。                   | ???              |
| `原始页面`       | 文章的原始内容                                       | ???              |
| `照片`         | 文章的照片（用于相簿）                                   | array            |
| `链接`         | 文章的外部链接（用于链接文章）                               | `字符串`            |

**文章 (`post`):** 与 `page` 布局相同，但新增以下变量。

| 变量     | 描述               | 类型                          |
| ------ | ---------------- | --------------------------- |
| `已发布`  | 如果帖子不是草稿，则为 true | `boolean`                   |
| `页面分类` | 该文章的所有分类         | `array`，包含了站点全部的标签 `object` |
| `标签`   | 该文章的所有标签         | `object` 或 `null`           |

**首页（`index`）**

| 变量      | 描述                                                       | 类型    |
| ------- | -------------------------------------------------------- | ----- |
| `页面`    | 每页显示的文章数量                                                | `数字`  |
| `总页面`   | 总页数                                                      | `数字`  |
| `当前页面`  | 目前页数                                                     | `数字`  |
| `当前网址`  | 目前分页的网址                                                  | `字符串` |
| `帖子`    | 本页文章 ([Data Model](https://hexojs.github.io/warehouse/)) | `对象`  |
| `上一页`   | 上一页的页数。 如果此页是第一页的话则为 `0`。                                | `数字`  |
| `前面的链接` | 上一页的网址。 如果此页是第一页的话则为 `''`。                               | `字符串` |
| `下一页`   | 下一页的页数。 如果此页是最后一页的话则为 `0`。                               | `数字`  |
| `下一页链接` | 下一页的网址。 如果此页是最后一页的话则为 `''`。                              | `字符串` |
| `路径`    | 当前页面的路径（不含根目录）。 我们通常在主题中使用 `url_for(page.path)`。         | `字符串` |

**分类 (`category`)** ：与 `index` 布局相同，但新增以下变量。

| 变量   | 描述               | 类型        |
| ---- | ---------------- | --------- |
| `存档` | 等于 `true`        | `boolean` |
| `年`  | 年份归档 (4位)        | `数字`      |
| `月`  | 月份归档 (没有前导零的2位数) | `数字`      |

**标签 (`tag`)** ：与 `index` 布局相同，但新增以下变量。

| 变量    | 描述   | 类型    |
| ----- | ---- | ----- |
| `类别：` | 分类名称 | `字符串` |

**归档 (`archive`)** ：与 `index` 布局相同，但新增以下变量。

| 变量   | 描述   | 类型    |
| ---- | ---- | ----- |
| `标签` | 标签名称 | `字符串` |

[Moment.js]: http://momentjs.com/
