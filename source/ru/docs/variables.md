### Глобальные переменные

| Переменная  | Описание                                                        | Тип                                   |
| ----------- | --------------------------------------------------------------- | ------------------------------------- |
| `сайт`      | Информация о сайте.                                             | `объект`; см. [Переменные сайта][]    |
| `страница`  | Информация о конкретной странице и переменные в шапке страницы. | `объект`; см. [переменные страницы][] |
| `настройки` | Конфигурация сайта.                                             | `объект` (_config файл вашего сайта)  |
| `тема`      | Конфигурация темы. Наследуется от конфигурации сайта.           | `объект` (файл _config вашей темы)    |
| `путь`      | Путь текущей страницы.                                          | `строка`                              |
| `url`       | Полная URL ссылка на текущую страницу.                          | `строка`                              |
| `env`       | Переменные среды.                                               | ???                                   |

{% note warn %}
Lodash удалён из глобальных переменных начиная с Hexo 5.0.0. [You-Dont-Need-Lodash-Underscore](https://github.com/you-dont-need/You-Dont-Need-Lodash-Underscore) might be helpful for your migration.
{% endnote %}

### Переменные сайта

| Переменная       | Описание      | Тип                       |
| ---------------- | ------------- | ------------------------- |
| `посты сайта`    | Все посты     | `array` of `post` objects |
| `сайт.страницы`  | Все страницы  | `array` of `page` objects |
| `сайт.категории` | Все категории | `array` of ???            |
| `теги сайта`     | Все теги      | `array` of ???            |

### Переменные страницы

**Статья (`страница`)**

| Переменная            | Описание                                                                                         | Тип                                               |
| --------------------- | ------------------------------------------------------------------------------------------------ | ------------------------------------------------- |
| `page.title`          | Название статьи                                                                                  | `строка`                                          |
| `дата`                | Дата создания статьи                                                                             | Дата последнего обновления (Объект [Moment.js][]) |
| `страница.обновлена`  | Дата последнего обновления статьи                                                                | Дата создания страницы (Объект [Moment.js][])     |
| `page.comments`       | Включены ли комментарии                                                                          | `boolean`                                         |
| `макет страницы`      | Имя макета                                                                                       | `строка`                                          |
| `контент страницы`    | Полностью обработанное содержание страницы                                                       | `строка`                                          |
| `page excerpt`        | Статья выдержки                                                                                  | `строка`                                          |
| `страница.подробнее`  | Содержимое страницы без отрывка                                                                  | `строка`                                          |
| `page.source`         | Путь к исходному файлу                                                                           | `строка`                                          |
| `page.full_source`    | Полный путь к исходному файлу                                                                    | `строка`                                          |
| `page.path`           | URL текущей страницы без корневой части адреса. Обычно используется `url_for(page.path)` в теме. | `строка`                                          |
| `page.permalink`      | Полный URL-адрес страницы                                                                        | `строка`                                          |
| `Предыдущая страница` | Предыдущий пост. `null`, если пост первый.                                                       | ???                                               |
| `страница.след.`      | Следующий пост. `null`, если пост последний.                                                     | ???                                               |
| `page.raw`            | Сырье данных статьи                                                                              | ???                                               |
| `page.photos`         | Фотографии из страницы (Используется в галерее постов)                                           | массив ???                                        |
| `страница.ссылка`     | Внешняя ссылка на статью (Используется в ссылках поста)                                          | `строка`                                          |

**Пост (post)**: такие же переменные, как и у страницы, но добавлены следующие переменные.

| Переменные              | Описание                                     | Тип            |
| ----------------------- | -------------------------------------------- | -------------- |
| `страница опубликована` | Верно, если сообщение не является черновиком | `boolean`      |
| `page.categories`       | Все категории поста                          | `array` of ??? |
| `теги страницы`         | Все теги поста                               | `array` of ??? |

**Главная страница (index)**

| Переменная            | Описание                                                                                         | Тип      |
| --------------------- | ------------------------------------------------------------------------------------------------ | -------- |
| `page.per_page`       | Количество постов, отображаемых на странице                                                      | `число`  |
| `итог страницы`       | Общее число постов                                                                               | `число`  |
| `страница.текущая`    | Номер текущей страницы                                                                           | `число`  |
| `page.current_url`    | URL текущей страницы                                                                             | `строка` |
| `page.posts`          | Посты на этой странице ([Data Model])                                                            | `объект` |
| `Предыдущая страница` | Номер предыдущей страницы. `0`, если текущая страница является первой.                           | `число`  |
| `page.prev_link`      | Ссылка на предыдущую страницу. Равна `''`, если текущая страница является первой.                | `строка` |
| `страница.след.`      | Номер следующей страницы. Равно `0`, если текущая страница является последней.                   | `число`  |
| `page.next_link`      | Ссылка на следующую страницу. Равно `''`, если текущая страница является последней.              | `строка` |
| `page.path`           | URL текущей страницы без корневой части адреса. Обычно используется `url_for(page.path)` в теме. | `строка` |

**Категория (category):** Такая же, как макет `index`, но добавлены следующие переменные.

| Переменная       | Описание                                     | Тип       |
| ---------------- | -------------------------------------------- | --------- |
| `страница.архив` | Приравнивается к `true`                      | `boolean` |
| `год.страница`   | Год архива (4-х значное)                     | `число`   |
| `месяц.страница` | Месяц архива (2-х значное без ведущих нулей) | `число`   |

**Тег (tag):** Такой же, как макет `index`, но добавлены следующие переменные.

| Переменная      | Описание      | Тип      |
| --------------- | ------------- | -------- |
| `page.category` | Имя категории | `строка` |

**Архив (archive):** Такой же, как макет `index`, но добавлены следующие переменные.

| Переменная     | Описание | Тип      |
| -------------- | -------- | -------- |
| `тэг страницы` | Имя тега | `строка` |
