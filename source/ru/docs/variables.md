### Глобальные переменные

| Переменная  | Описание                                                                      | Тип                                   |
| ----------- | ----------------------------------------------------------------------------- | ------------------------------------- |
| `сайт`      | Информация по всему сайту.                                                    | `объект`; см. [Переменные сайта][]    |
| `страница`  | Конкретная информация о страницах и настраиваемые переменные в переднем крае. | `объект`; см. [переменные страницы][] |
| `настройки` | Конфигурация сайта.                                                           | `объект` (_config файл вашего сайта)  |
| `тема`      | Конфигурация шаблона. Наследует от конфигурации сайта.                        | `объект` (файл _config вашей темы)    |
| `путь`      | Путь к текущей странице                                                       | `строка`                              |
| `url`       | Полный URL текущей страницы                                                   | `строка`                              |
| `env`       | Переменные окружения                                                          | ???                                   |

{% note warn %}
Lodash удалён из глобальных переменных начиная с Hexo 5.0.0. [You-Dont-Need-Lodash-Underscore](https://github.com/you-dont-need/You-Dont-Need-Lodash-Underscore) might be helpful for your migration.
{% endnote %}

### Переменные страницы

**Статья (`страница`)**

| Переменная            | Описание                                                                        | Тип                  |
| --------------------- | ------------------------------------------------------------------------------- | -------------------- |
| `page.title`          | Название статьи                                                                 | `строка`             |
| `дата`                | Дата создания статьи                                                            | [Moment.js][] объект |
| `страница.обновлена`  | Дата последнего обновления статьи                                               | [Moment.js][] объект |
| `page.comments`       | Комментарий включен или не включен                                              | `boolean`            |
| `макет страницы`      | Название макета                                                                 | `строка`             |
| `контент страницы`    | Полное обработанное содержимое статьи                                           | `строка`             |
| `page excerpt`        | Статья выдержки                                                                 | `строка`             |
| `страница.подробнее`  | Содержание за исключением выдержки из статьи                                    | `строка`             |
| `page.source`         | Путь в исходном файле                                                           | `строка`             |
| `page.full_source`    | Полный путь исходного файла                                                     | `строка`             |
| `page.path`           | URL статьи без корневого URL. Мы обычно используем `url_for(page.path)` в теме. | `строка`             |
| `page.permalink`      | Полный (закодированный) URL статьи                                              | `строка`             |
| `Предыдущая страница` | The previous post, `null` if the post is the first post                         | ???                  |
| `страница.след.`      | Следующий пост, `null` если сообщение является последним сообщением             | ???                  |
| `page.raw`            | Сырье данных статьи                                                             | ???                  |
| `page.photos`         | Фото статьи (используется в галереях записей)                                   | массив ???           |
| `страница.ссылка`     | Внешняя ссылка статьи (используется в сообщениях ссылки)                        | `строка`             |

**Пост (`post`):** То же, что и макет страницы `` но добавьте следующие переменные.

| Переменная              | Описание                                     | Тип            |
| ----------------------- | -------------------------------------------- | -------------- |
| `страница опубликована` | Верно, если сообщение не является черновиком | `boolean`      |
| `page.categories`       | Все категории поста                          | `array` of ??? |
| `теги страницы`         | Все теги поста                               | `array` of ??? |

**Главная (`индекс`)**

| Переменная            | Описание                                                                                  | Тип      |
| --------------------- | ----------------------------------------------------------------------------------------- | -------- |
| `page.per_page`       | Сообщения, отображаемые на странице                                                       | `число`  |
| `итог страницы`       | Общее количество страниц                                                                  | `число`  |
| `страница.текущая`    | Текущий номер страницы                                                                    | `число`  |
| `page.current_url`    | URL текущей страницы                                                                      | `строка` |
| `page.posts`          | Сообщения на этой странице ([Модель данных](https://hexojs.github.io/warehouse/))         | `объект` |
| `Предыдущая страница` | Номер предыдущей страницы. `0` если текущая страница является первым.                     | `число`  |
| `page.prev_link`      | URL предыдущей страницы. `''` если текущая страница является первым.                      | `строка` |
| `страница.след.`      | Номер следующей страницы. `0` если текущая страница является последним.                   | `число`  |
| `page.next_link`      | URL следующей страницы. `''` если текущая страница является последним.                    | `строка` |
| `page.path`           | URL текущей страницы без корневого URL. Мы обычно используем `url_for(page.path)` в теме. | `строка` |

**Archive (`archive`):** Same as `index` layout but add the following variables.

| Переменная       | Описание                                      | Тип       |
| ---------------- | --------------------------------------------- | --------- |
| `страница.архив` | Equals `true`                                 | `boolean` |
| `год.страница`   | Год архива (4-значный)                        | `число`   |
| `месяц.страница` | Месяц архива (2-значный без лидирующих нулей) | `число`   |

**Category (`category`):** Same as `index` layout but add the following variables.

| Переменная      | Описание           | Тип      |
| --------------- | ------------------ | -------- |
| `page.category` | Название категории | `строка` |

**Tag (`tag`):** Same as `index` layout but add the following variables.

| Переменная     | Описание | Тип      |
| -------------- | -------- | -------- |
| `тэг страницы` | Имя тега | `строка` |
