---
title: Развертыватель
---

Позволяет быстро развернуть свой сайт на удалённом сервере без сложных команд.

## Краткий обзор

``` js
hexo.extend.deployer.register(name, function(args){
  // ...
});
```

В аргумент `args` передаётся функция, которая содержит развернутое значение, взятое из `_config.yml`. Он содержит значение `deploy`, установленное в `_config.yml`, а также точные данные, которые пользователи вводили в свой терминал.
